###########################################     HOW TO     ####################################################


Before running the program, please install all the requirements. This can be installed by the following running command:

		pip install -r requirements.txt

in order to run the game, use following command in the directory with the modules:

		python3 playSnakeGame.py --[mode]

differents modes in this project are:-
	1. keys
	2. shortest
	3. beter-shortest
	4. hamiltonian
if something other than the above modes are used the program will automatically choose the KEYS mode
